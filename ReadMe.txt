Zailab Acme Bank maven project consists of a Springboot backend (zailab-accounts-backend V1.0.0) and frontend (zailab-accounts-app V1.0.0)

1. First build the frontend by doing a maven clean then install.

2. Build the backend by also doing a maven clean then install.

3. Navigate to /zailab-accounts-backend

4. Run java -jar target/zailab-accounts-backend-1.0.0.war

5. Open this url http://localhost:8080/#/login or http://localhost:8080/ on browser, make sure you have the lastest browsers.

6. login details username = user, password = password for all login in screens.
